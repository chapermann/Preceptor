# Preceptor
Filosofia do Preceptor  O Preceptor não é um prontuário eletrônico.  O Preceptor não substitui o médico.  O Preceptor é um copiloto clínico.  Seu objetivo é organizar o raciocínio médico, reduzir esquecimentos, estruturar a anamnese e o exame físico, apoiar a elaboração de hipóteses diagnósticas e produzir documentação técnica de alta qualidade.
